# IT外包服务网站项目大纲

## 文件结构

### HTML页面 (4个页面)
1. **index.html** - 主页
   - Hero区域：动态背景 + 公司介绍
   - 服务概览：6个核心服务卡片
   - 服务配置器：主要交互功能（左侧服务选择，中间配置选项，右侧报价）
   - 监控面板：展示技术实力的实时数据
   - 客户案例：轮播展示成功案例
   - 联系表单：快速咨询入口

2. **services.html** - 服务详情页
   - 各服务详细介绍
   - 技术规格对比表
   - 服务流程说明
   - 价格套餐展示

3. **about.html** - 关于我们页
   - 公司介绍和愿景
   - 团队展示（使用卡片布局）
   - 发展历程时间线
   - 资质证书展示

4. **contact.html** - 联系页面
   - 联系方式信息
   - 在线表单（姓名、公司、需求、预算等）
   - 地图位置（使用Leaflet）
   - 常见问题FAQ

### 资源文件
- **resources/** - 资源文件夹
  - hero-bg.jpg - Hero区域背景图
  - server-icon.svg - 服务器图标
  - website-icon.svg - 网站建设图标
  - email-icon.svg - 企业邮箱图标
  - security-icon.svg - 安防监控图标
  - maintenance-icon.svg - 运维服务图标
  - vps-icon.svg - VPS服务图标
  - team-*.jpg - 团队成员照片（搜索或生成）
  - case-*.jpg - 案例展示图片（搜索或生成）

### JavaScript文件
- **main.js** - 主要交互逻辑
  - 服务配置器逻辑
  - 监控数据模拟
  - 轮播控制
  - 聊天窗口功能
  - 表单验证
  - 动画控制

## 核心功能模块

### 1. 服务配置器 (主要交互功能)
- 服务选择面板：6个主要服务类别
- 配置选项：根据选择的服务显示不同配置项
- 实时报价：动态计算服务价格
- 联系表单：收集客户需求信息

### 2. 监控面板
- 服务器状态监控（CPU、内存、网络）
- 网络设备状态
- 实时数据更新动画
- 点击查看详细信息

### 3. 案例展示轮播
- 自动轮播展示
- 手动控制
- 项目详情展示

### 4. 在线客服
- 浮动聊天图标
- 预设问题快速回复
- 智能问答系统

## 技术实现

### 使用的库
- Anime.js - 动画效果
- ECharts.js - 数据可视化
- p5.js - 粒子背景效果
- Splide.js - 轮播组件
- Matter.js - 物理动画
- PIXI.js - 高性能渲染
- Shader-park - 着色器效果
- Leaflet - 地图展示

### 样式框架
- Tailwind CSS - 响应式布局
- 自定义CSS - 特殊效果

### 数据结构
- 服务配置数据（价格、规格等）
- 监控模拟数据
- 案例展示数据
- 团队成员信息

## 页面导航
- 顶部导航栏：首页、服务、关于我们、联系我们
- 底部导航：快速链接、联系方式、社交媒体
- 移动端：汉堡菜单

## 响应式设计
- 桌面端：1200px+ 充分利用宽屏
- 平板端：768px-1199px 适配中等屏幕
- 移动端：<768px 垂直布局
