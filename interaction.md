# IT外包服务网站交互设计

## 核心交互功能

### 1. 服务配置器与报价系统
- **功能描述**: 用户可以选择所需的服务组合，实时计算价格
- **交互流程**:
  - 左侧：服务类别选择面板（网站建设、服务器、VPS、邮箱、运维、安防）
  - 中间：服务选项和配置（根据选择的类别显示不同选项）
  - 右侧：实时报价单和联系表单
- **数据内容**:
  - 网站建设：企业官网、电商网站、定制开发，每个包含不同功能模块
  - 服务器出租：CPU核心数、内存、硬盘、带宽等配置选项
  - VPS：不同规格的虚拟服务器套餐
  - 企业邮箱：用户数量、存储空间、安全功能
  - 网络运维：服务等级、响应时间、覆盖范围
  - 监控安防：摄像头数量、存储时长、智能分析功能

### 2. 服务状态监控面板
- **功能描述**: 展示服务器和网络设备的实时状态（模拟数据）
- **交互流程**:
  - 网格布局显示多个服务器/设备卡片
  - 每个卡片显示：设备名称、状态指示灯、CPU使用率、内存使用率、网络流量
  - 点击卡片显示详细信息和历史数据图表
- **动态效果**: 实时更新的数据、状态指示灯闪烁、进度条动画

### 3. 案例展示轮播
- **功能描述**: 展示公司完成的IT项目案例
- **交互流程**:
  - 自动轮播展示不同行业的成功案例
  - 每个案例包含：项目名称、行业、服务内容、技术栈、效果截图
  - 用户可以点击暂停自动轮播，手动浏览

### 4. 在线客服聊天窗口
- **功能描述**: 模拟在线客服咨询系统
- **交互流程**:
  - 右下角固定的聊天图标
  - 点击打开聊天窗口
  - 预设常见问题快速回复
  - 用户输入问题后显示智能回复

## 页面结构

### 主页 (index.html)
- Hero区域：动态背景 + 公司介绍 + CTA按钮
- 服务概览：6个核心服务的图标卡片
- 服务配置器：主要交互功能
- 监控面板：展示技术实力
- 客户案例：轮播展示
- 联系表单

### 服务详情页 (services.html)
- 各服务的详细介绍和价格表
- 技术规格对比
- 服务流程说明

### 关于我们页 (about.html)
- 公司介绍
- 团队展示
- 发展历程
- 资质证书

### 联系页面 (contact.html)
- 联系方式
- 在线表单
- 地图位置
- 常见问题
